# Quiz-Application
An online quiz application with multiple categories and timer.
Features:
- Multiple categories (GK, Math, Science, English)
- Timer-based quiz (up to 60 minutes)
- Randomized questions
- Score calculation and scoreboard
- Responsive UI

Tech Stack:
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)

This project was built as part of my learning to strengthen full-stack development skills.
